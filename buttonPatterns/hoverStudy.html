1. element:hover

Indicates that when the mouse hovers over this element, the style of this element will change.

2. element:hover element

Indicates that when the mouse hovers over the parent element, the style of its child elements will change.

Hover over this text

Child element changes color when parent is hovered.

3. element:hover + element

Indicates that when the mouse hovers over an element, the style of its adjacent sibling element will change.

Hover over this heading

First paragraph changes color when heading is hovered.

4. element:hover ~ element

Indicates that when the mouse hovers over an element, the style of all its subsequent sibling elements will change, regardless of whether these sibling elements are adjacent.

Hover over this heading

First paragraph changes color when heading is hovered.

Second paragraph also changes color when heading is hovered.

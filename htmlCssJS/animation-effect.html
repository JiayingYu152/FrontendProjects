Animation based on CSS

Jumping Heart

  • Using animation: piece-1 3s infinite; to define the animation-name, animation-duration, animation-effect, respectively.
  • Using :nth-child(#) to define which piece is selected.
  • Using @keyframes to define the animation effect, NAME your animation.
  • UI TIPS: using border-radius: # makes your interface more friendly and soft.

See the Pen Jumping Heart by Jiaying Yu (@JiayingYu152) on CodePen.

Vertical Bounce

  • Using background: linear-gradient(150deg, #d299c2, #f2f9d7) to create a linear gradient background. REMEMBER: it is background instead of background-color. In this case, I use this field for both the bubble and the whole page background.
  • Using @keyframes to define the animation effect, NAME your animation.
  • Using animation: ANIMATION-NAME #s infinite; to define the animation-name, animation-duration, animation-effect, respectively.
  • Using filter: blur(1px) to add a blur effect, making your item more soft. The greater the value, the more blur the item will be.

See the Pen Vertical Bounce by Jiaying Yu (@JiayingYu152) on CodePen.

Authentic Drip

  • IMPORTANT: Using CSS Border Radius Generator to draw the border-radius for the animation.
  • Check out #water {box-shadow: XXX} to see how I draw the water drip to make it look like a 3D effect.
  • Using @keyframes to change various border-radius to make the water drip move.

Raising Background

  • Using :nth-child(#) to define which child is selected.
  • Using @keyframes to define the animation effect, NAME your animation.
  • CODING TIPS: in VSCode(that I usually use), type li*21 then ENTER, you will create 21 "li" immediately. Same as this tip, if you want to create multiple FIELD with the same className, you can type item#className*repeatTime, like(li#myLi*2). Try it on, and you will be surprised!
